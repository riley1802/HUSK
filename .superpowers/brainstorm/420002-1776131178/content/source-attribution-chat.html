Attribution Design Details

  • Sources panel attaches directly below the response bubble (2px top radius creates visual continuity)
  • Collapsed by default — shows "Sources · N documents" toggle. Tap to expand.
  • Sand accent bar (#D6C9A8) on highest-relevance source, muted accent (#9C9276) on lower
  • Relevance score shown as percentage — accent-colored for high, muted for lower
  • Chunk preview in muted text (#6B7280) — truncated to 2 lines
  • No sources case — when nothing retrieved above threshold, no sources section appears at all
  • User bubble uses existing userBubbleBgColor (#9C9276 / accentMuted)
  • Agent bubble uses existing agentBubbleBgColor (#1B1F24 / surfaceHigh)
