Design Notes

  • Background: #0E1013 — matches HUSK's huskBackground
  • Cards: #15181C (huskSurface) with 12dp corner radius — same as model item cards
  • Elevated surfaces: #1B1F24 (surfaceHigh) — for stats bar and chunk previews
  • Accent: #D6C9A8 (sand) — used on the Add Document button and active embedding label
  • Status colors: Success #7E9B7A, Warning #C9A86E, Error #C97A6E — HUSK's muted palette
  • Typography: Nunito, same weight hierarchy — 600 for titles, 700 for labels, 400 for body
  • Spacing: 24dp horizontal padding, 16dp card padding, 8dp card gaps — matches HomeScreen
  • Progress bar: Sand accent on outline (#2C313A) track — same as download progress pattern
